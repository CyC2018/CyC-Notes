## cycle sort + scan array
这篇文章很赞：https://emre.me/coding-patterns/cyclic-sort/

time:O(N)
space:O(1)
```java
class Solution {
    public int missingNumber(int[] nums) {
        int start = 0;
        // 开始循环
        while (start < nums.length) {
            // 找到当前值，准备把它放到该放的位置去
            int target = nums[start];
            // 如果值和位置不对应，就不停地把值跟它该对应位置上的值交换，如果值和位置对应则继续前进
            // 其实这里target的值也有可能为nums.length，但是找不到对应位置，所以就不管它跳过
            // 最后它的位置其实就是缺失数的位置
            if (target < nums.length && target != start) {
                swap(nums, target, start);
            } else {
                start++;
            }
        }
        // 最后扫描一遍数组，计数对应的个数，第一个发现不对应的地方就是缺失数的地方
        int res = 0;
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] == i) res++;
            else break;
        }
        return res;
    }
    private void swap(int[] nums, int i, int j) {
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}
```

## math(calculate the sum from 0 to n, then delete all values from the given array)
```java
class Solution {
    public int missingNumber(int[] nums) {
        int n = nums.length;
        return (1 + n) * n / 2 - Arrays.stream(nums).sum();
    }
}
```

## xor
```java
class Solution {
    public int missingNumber(int[] nums) {
        int res = 0;
        for (int i = 0; i < nums.length; i++) {
            res ^= i ^ nums[i];
        }
        return res ^= nums.length;
    }
}
```

## binary search
```java
class Solution {
    public int missingNumber(int[] nums) {
        Arrays.sort(nums);
        int left = 0, right = nums.length - 1;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (nums[mid] == mid) left = mid + 1;
            else right = mid - 1;
        }
        if (left >= nums.length && nums[right] == right) return nums[right] + 1;
        if (right < 0 && nums[left] != left) return nums[left] - 1;
        return nums[right] + 1;
    }
}
```
